# paper-bb-aid-mlm

Reproducible pipeline for an academic paper combining:

- **Balkan Barometer** survey data (SPSS extracts)
- **GODAD geocoded aid projects** (multi-donor)
- **AidData China** geocoded projects
- **AidData GeoQuery covariates** (e.g., night lights, population)
- Construction of a **nested/multilevel analysis dataset** (`dt`) with centered continuous variables
- **Descriptive statistics + visuals** and **multilevel models**

**Main language:** R  
**Main working dataset object:** `dt`
